# Agents

This directory stores reusable `AGENTS.md` templates and related agent-guidance patterns.

Use this area for:

- baseline `AGENTS.md` templates for new projects
- variants tuned for specific repository types or delivery contexts
- experiments in agent operating guidance that may later be standardized

Templates here should favor clarity, reuse, and explicit tradeoffs over project-specific assumptions.

Start from [`AGENTS.template.md`](D:/Projects/agoge/agents/AGENTS.template.md) when creating a new reusable agent guide.

## Concrete Roles

- [`business-analyst.md`](D:/Projects/agoge/agents/business-analyst.md) defines a structured interviewer role for early project discovery, business requirements, and process analysis.

Use the Business Analyst role when a project needs help clarifying business goals, documenting as-is and to-be processes, verifying requirements, maintaining traceability, and preparing downstream roles for solutioning without mixing in implementation ownership too early.
